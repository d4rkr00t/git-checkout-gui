## Install

```sh
npm install -g git-checkout-gui
```

## Usage

```sh
git checkout-gui
```

Or you can add alias to your .gitconfig:
```sh
# .gitconfig
[alias]
  cog = checkout-gui

# shell
git cog
```
Example .gitconfig — [d4rkr00t/dotfiles/../.gitconfig#L15](https://github.com/d4rkr00t/dotfiles/blob/master/link/.gitconfig#L15).
